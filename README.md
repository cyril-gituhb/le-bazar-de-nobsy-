# 🎮 Vancyr Ludo

Un jeu web interactif simple et amusant créé avec HTML, CSS et JavaScript.

## 📋 Description

Vancyr Ludo est un jeu de clicker addictif où vous devez cliquer sur le bouton autant que possible pour augmenter votre score et progresser de niveau.

## 🎯 Fonctionnalités

- ✅ Interface moderne et responsive
- ✅ Système de scoring en temps réel
- ✅ Progression par niveaux
- ✅ Difficulté progressive (plus de clics nécessaires pour passer au niveau suivant)
- ✅ Notifications de succès
- ✅ Réinitialisation du jeu
- ✅ Responsive design (fonctionne sur mobile, tablette et desktop)

## 🚀 Démarrage rapide

1. Clonez le dépôt :
```bash
git clone https://github.com/cyril-gituhb/vancyr-ludo.git
```

2. Ouvrez `index.html` dans votre navigateur :
```bash
cd vancyr-ludo
open index.html  # macOS
# ou
start index.html # Windows
# ou
xdg-open index.html # Linux
```

## 🎮 Comment jouer

1. Cliquez sur le bouton **"Démarrer"** pour commencer le jeu
2. Cliquez sur le bouton **"Cliquez ici!"** autant que possible
3. Chaque clic augmente votre score
4. Progressez de niveau tous les 10 clics au niveau 1, 20 clics au niveau 2, etc.
5. Utilisez **"Réinitialiser"** pour recommencer
